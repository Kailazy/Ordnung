Ordnung

A fast, self-sufficient DJ music catalog, analyzer, and rekordbox / CDJ USB exporter — written in Rust.

Download for macOS GitHub

Universal .dmg — Apple Silicon + Intel · macOS 11+

What it does

♪
Catalog your libraryScans your music into a fast SQLite catalog — the single source of truth, with duplicate and missing-file detection.
◷
Analyze everythingTempo, beatgrids, key, waveforms and loudness — computed on its own with pure-Rust decoders.
◆
Camelot keysDetects musical key and renders it as Camelot by default, so harmonic mixing is built in.
8A8B9A5A12B
⇲
rekordbox / CDJ exportGenerates a native rekordbox USB the CDJs can read — export.pdb and ANLZ analysis — without rekordbox itself.
⛉
Source files are sacredTag writes and conversions are opt-in. Deletes go to the OS Trash — never a hard remove.

Install

  1. Download the latest .dmg and open it.
  2. Drag Ordnung into your Applications folder.
  3. First launch only: right-click Ordnung.app → Open → Open.

The one-time right-click is needed because the build is ad-hoc signed, not notarized by Apple — macOS warns once on apps from unidentified developers, then trusts it. Or run xattr -dr com.apple.quarantine /Applications/Ordnung.app.
