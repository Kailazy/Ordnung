Ordnung

A fast, self-sufficient DJ music catalog, analyzer, and rekordbox / CDJ USB exporter — written in Rust.

Download for macOS View on GitHub

Universal macOS .dmg — Apple Silicon + Intel. Requires macOS 11+.

What it does

♪

Catalog your library

Scans your music and builds a fast SQLite catalog — the single source of truth, with duplicate and missing-file detection.

◷

Analyze everything

Tempo, beatgrids, musical key, waveforms and loudness — computed on its own with pure-Rust decoders.

◆

Camelot keys

Detects musical key and renders it as Camelot by default, so harmonic mixing is built into your catalog.

⇲

rekordbox / CDJ export

Generates a native rekordbox USB the CDJs can read — export.pdb and ANLZ analysis — without rekordbox itself.

⌘

Native desktop app

A fast egui app with an interactive waveform and zoom lane. A matching CLI exposes the same engine for scripting.

⛉

Source files are sacred

Tag writes and conversions are opt-in. Deletes go to the OS Trash — never a hard remove. Your originals stay untouched.

Install

  1. Download the latest .dmg and open it.
  2. Drag Ordnung into your Applications folder.
  3. First launch only: right-click Ordnung.app → Open → Open.

The one-time right-click is needed because the build is ad-hoc signed, not notarized by Apple — macOS Gatekeeper warns once on apps from unidentified developers, then trusts it from then on. You can also run xattr -dr com.apple.quarantine /Applications/Ordnung.app.
